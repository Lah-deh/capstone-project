Overview
This project is a mobile-first web application built as a capstone project to strengthen core frontend skills using HTML, CSS, and JavaScript. The focus of the project is on layout structure, styling, and interactive behavior within a mobile viewport.
The interface follows a mobile-only design specification provided by the designer, with no desktop layout defined.

Features
Mobile-first layout
Structured HTML markup
CSS styling and layout positioning
JavaScript-driven interactivity
Clean user interface optimized for small screens

Technology Stack
HTML
CSS
JavaScript
GitHub Pages (deployment)

Design Scope
This project was implemented based on a mobile-only design specification. Desktop and large-screen layouts were not included in the original design and are intentionally excluded from this implementation.

What This Project Demonstrates
Strong understanding of HTML structure
CSS layout and styling fundamentals
JavaScript for user interaction
Ability to translate design specifications into functional interfaces
Mobile-first development approach

Notes
For the best experience, view this project on a mobile device or using a browser’s mobile viewport.
